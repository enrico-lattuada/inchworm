# inchworm
A units-and-quantities library, written in Rust

## Install Python lib from source

Run:

```
python -m pip install .
```

## Use `pre-commit`

1. Install `pre-commit` with
    ```
    pip install pre-commit
    ```
2. Install `pre-commit` hooks
    ```
    pre-commit install
    ```
3. (Optional) Run `pre-commit` on all files
    ```
    pre-commit run --all-files
    ```
